## Prompting rules

- Attach the correct reference image for each single-book slide; do not rely on title text alone.
- For the final CTA bundle, attach all references when the provider allows it. Ask for each book once, no extras, no duplicates, natural physical arrangement, visible page edges and contact shadows.
- Use the user's exact baked text, but verify because image generators frequently distort typography.
- Style recipe: dark marble/walnut, brass lamps, black silk, burgundy velvet, pearls, perfume bottles, quilted handbag, city-light bokeh, warm ivory/champagne serif typography.
- Strong negatives: no phone frame, screenshot UI, TikTok handle, black bars, collage, multiple panels, anime/illustration, plastic CGI, wrong book, extra book, altered cover, invented title/author.

## Delivery

After generation, verify dimensions. The successful outputs in this session were `1086 × 1448` true 3:4. Deliver each image with its label and path, and mention the reference mapping if the user asks why a reference was attached to a given prompt.
